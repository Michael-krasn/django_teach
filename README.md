# Интернет-магазин на Flask

Веб-приложение интернет-магазина с использованием Flask (бэкенд) и Bootstrap 5 (фронтенд). Проект включает главную страницу, каталог товаров, страницы категорий и форму обратной связи.

## 🚀 Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone <url-репозитория>

# 2. Перейдите в директорию проекта
cd project

# 3. Создайте виртуальное окружение
python -m venv venv

# 4. Активируйте виртуальное окружение
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 5. Установите зависимости
pip install -r requirements.txt

# 6. Запустите приложение
python app.py
```

После запуска откройте браузер и перейдите по адресу: http://127.0.0.1:5000/
